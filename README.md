# 🍃 Komorebi (木漏れ日) - 智能多媒体处理引擎

**Komorebi**（木漏れ日），意为“穿透树叶的阳光”。
本项目致力于在多媒体处理领域中寻找那道完美的光芒——**在“高保真画质”、“极致缩减的体积”与“极限的处理速度”之间，实现优雅的平衡。**

无需繁琐的命令行，无需配置 Python 或 FFmpeg 环境变量。Komorebi 现已进化为现代化的独立桌面应用，为您提供极客级别的视音频压制与转换体验。

---

## ✨ 核心特性 (Features)

### 🖥️ 现代化极简 GUI 与开箱即用
* **零环境依赖**：将庞大的 Python 运行库与 FFmpeg 核心引擎完美封装。只需下载安装包，一键安装，即刻使用。
* **暗黑极客主题**：沉浸式的深色 UI 设计，丝滑的操作反馈，告别传统命令行的枯燥。

### 🎬 场景化极致压缩 (Scene-Optimized Compression)
不再是冷冰冰的比特率参数，针对不同视频源提供专属算法预设：
* **动漫 / 二次元**：专属去色带保护，强化线条锐利度。
* **电脑录屏 / 游戏**：极限压缩率，保护文字UI界面不发虚。
* **实拍视频 / 电影**：保留真实物理光影与噪点细节。
* *内置实时体积与画质损耗预估（基于混合 AI 算法的动态计算）。*

### 🔄 智能格式化工厂 (Smart Format Factory)
* **全格式兼容**：支持 `mp4`, `mkv`, `webm`, `gif`, `mp3`, `flac` 等数十种音视频与动图格式互转。
* **无损音频支持**：自动识别 FLAC/WAV，并提供动态防呆锁定。
* **外部音轨挂载**：智能侦测无声视频，自动解锁并支持混入外部音轨/背景音乐。

### 🚀 强劲的并发与算力分配 (Hardware Acceleration & Batching)
* **全平台硬件加速**：自动侦测并调用 NVIDIA (NVENC)、Intel (QSV) 或 AMD (AMF) 显卡进行极限加速。
* **高级任务队列**：支持拖入整个文件夹。允许应用全局设置，**或为列表中的每一个文件单独定制编码与画质参数**。
* **算力自定义**：支持手动分配“并发任务数”与“单任务线程限制”，完美适配从轻薄本到高性能 NAS 的各类物理核心。

### 🛡️ 军工级防崩容灾系统 (Auto-Fallback System)
* **智能纠错拦截**：如遇纯音频转MP4、WebM容器拒载H.264等物理冲突，系统将在毫秒级静默修正，防止底层报错。
* **热切换重试**：当显卡显存爆满或硬件解码器崩溃时，自动热切换至纯 CPU 高保真软压制继续任务。
* **紧急制动**：一键安全终止所有排队与运行中的底层进程，杜绝僵尸进程残留。

## 📦 安装说明 (Installation)

1. 在 [Releases] 页面下载最新版本的 `Komorebi_Installer_v1.0.exe`。
2. 双击运行安装包，跟随向导选择您心仪的安装目录。
3. 安装完成后，在桌面双击 **Komorebi** 图标即可启动。
> **注**：本软件已内置所有必要的环境，您的电脑无需提前安装 Python 或 FFmpeg。

---

## 🛠️ 快速入门 (Quick Start)

1. **导入媒体**：点击“选文件”或“选目录”导入需要处理的音视频源。
2. **选择模式**：
   * **模式 1 (场景压缩)**：适合需要大幅瘦身但肉眼看不出画质降低的视频归档。
   * **模式 2 (格式工厂)**：适合跨设备兼容性转换或提取无损音频。
3. **参数配置**：根据界面下拉菜单的“体积预估”选择合适的预设挡位。
4. **启动引擎**：点击【🚀 启动核心引擎】，并在黑色终端框内实时监控底层处理进度。
