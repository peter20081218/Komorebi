## 📖 名字的由来 (Origin)

**Komorebi** 取自日语“**木漏れ日**”，意为“从树叶缝隙中透过的阳光”。

正如其名，本脚本旨在通过精细的参数筛选，滤除冗余的视频数据，在**画质（Fidelity）**、**体积（Size）**与**速度（Speed）**之间找到那个完美的**平衡点 P**。在 FFmpeg 那幽暗、繁杂且令人望而生畏的命令行森林中，Komorebi 为用户指引出一条高效、透明且优雅的自动化处理之路。

## ✨ 核心功能全览 (Full Features)

### 1. 🎬 场景化高保真压制
针对不同类型的片源内置了大师级算法预设，确保在最低码率下保留最高画质：
* **动漫/二次元模式**：自动套用去色带与线条锐利保护滤镜，防止压制后出现色块。
* **录屏/游戏模式**：保护文字边缘不发虚，动态调整关键帧间隔，获得极高压缩率。
* **实拍/电影模式**：保留真实噪点与物理光影细节，防止画面出现“塑料感”。

### 2. 🚀 全链路硬件加速引擎
* **智能探测**：自动识别 NVIDIA (NVENC)、Intel (QSV)、AMD (AMF) 硬件加速接口。
* **全速驱动**：开启“硬件解码 + 硬件编码”全链路加速，速度较传统软压提升数倍。

### 3. 🧠 智能容灾与自愈系统
* **自动变阵**：当遇到物理冲突（如 PCM 音频塞入 MP3 容器）时，系统自动拦截报错，热切换为高保真转码继续执行。
* **硬件回退**：若显卡编码器因意外或驱动问题瘫痪，自动撤销任务并无缝切换至纯 CPU 模式。

### 4. ⚙️ 系统级资源管理
* **算力隔离**：自动探测 CPU 物理核心，为每个并发任务分配专属线程数，保障系统操作不卡顿。
* **僵尸进程清理**：拦截 Windows 关闭信号，无论正常退出还是强制关闭窗口，均能确保所有 `ffmpeg` 进程被干掉。

### 5. 📊 极致的用户体验
* **步进式向导**：支持输入 `b` 返回上一步。
* **精准预估**：基于混合 AI 模型，在开始前即可预知预计体积与画质损失率。
* **全局进度**：批量任务下自动汇总时长，提供单一、整洁的总体完成进度条。

## 🛠️ 新手部署教程 (Getting Started)

### 第一步：配置 Python 环境
1.  前往 [Python 官方下载页面](https://www.python.org/downloads/) 下载最新安装包。
2.  **⚠️ 关键步骤**：安装时务必勾选底部界面的 **`Add Python to PATH`**。
3.  打开 CMD，输入 `python --version` 确认安装成功。

### 第二步：配置 FFmpeg 核心组件
1.  前往 [Gyan.dev](https://www.gyan.dev/ffmpeg/builds/) 下载 `ffmpeg-release-full.7z`。
2.  解压并将文件夹移动到 `C:\Program Files\FFmpeg`。
3.  将里面的 `bin` 文件夹路径添加到系统的**环境变量 Path** 中。
4.  打开 CMD，输入 `ffmpeg -version` 确认成功。

### 第三步：运行脚本
1.  下载本项目中的 `Komorebi.py`。
2.  在脚本所在目录打开终端，输入 `python Komorebi.py` 即可开启你的高效处理之旅。
